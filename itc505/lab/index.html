Addendum:I took on the task of using JavaScript to create an interactive counter in this project. The primary objective was to dynamically create a random number and show it on the website in response to user input.

Overview: The "Generate Number" button is clicked to activate the JavaScript code. The script creates a random number between 1 and 100 and places it in the specified spot on the page when the button is clicked.

Difficulties Encountered: I ran quite a few difficulties when putting this functionality into practice. Making sure the random number generated was really random and uniformly dispersed was one significant challenge. To solve this, I used the Math.random() function, modifying the range as necessary.

Solutions: I used the formula Math.floor(Math.random() * 100) + 1 to solve the randomization issue. The random integer between 1 and 100 is guaranteed by this formula. In order to make the webpage interactive and successfully respond to user activities, I also employed event listeners.

All in all, this project gave me a great chance to improve my knowledge of JavaScript and include dynamic features into webpages.
